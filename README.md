# Multi-Service Booking System

A web-based application that allows users to book multiple services online and enables admins to manage appointments efficiently.

## 🔧 Features
- User Registration & Login
- Service Booking System
- Admin Dashboard
- Appointment Management

## 💻 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## ⚙️ How to Run
1. Install XAMPP
2. Move project to `htdocs`
3. Import `database.sql` into phpMyAdmin
4. Start Apache & MySQL
5. Open in browser: http://localhost/Service-booking

## 📌 Note
This project runs locally using XAMPP and is not deployed online.
