# Multi-Service Booking System

This is a web-based application that allows users to book multiple services online.

## Features
- User registration and login
- Service booking
- Admin panel
- Appointment management

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How to Run
1. Install XAMPP
2. Move project to htdocs
3. Import database (.sql file)
4. Run localhost
